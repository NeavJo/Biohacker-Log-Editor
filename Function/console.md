# 控制台命令文档

## 触发方式

### 方式一：键盘快捷键（桌面端）

在页面中依次按下键盘 `C` → `M` → `D` 键，可打开控制台。

### 方式二：连续点击（移动端）

在移动端模式下，4秒内连续点击4次左上角的创建卡片按钮（黄色便签图标），可打开控制台。

---

## 操作说明

- **打开**：
  - 桌面端：按顺序按下 `C` `M` `D`
  - 移动端：4秒内点击左上角按钮4次
- **关闭**：按 `Escape` 键或点击控制台外部
- **发送命令**：输入命令后按 `Enter`

## 命令列表

### 1. create - 创建指定日期的卡片

**格式：**
```
create(yyyy.m.d)
```

**参数说明：**
- `yyyy`：4位年份（如 2024）
- `m`：月份（1-12）
- `d`：日期（1-31，需与月份匹配）

**校验规则：**
- 月份必须在 1-12 之间
- 日期必须符合对应月份的天数
  - 1,3,5,7,8,10,12 月：31天
  - 4,6,9,11 月：30天
  - 2 月：平年28天，闰年29天
  - 闰年规则：能被4整除且不能被100整除，或能被400整除

**示例：**
```
create(2024.6.15)   # 创建 2024年6月15日 的卡片
create(2024.2.29)   # 创建 2024年2月29日 的卡片（闰年有效）
create(2023.2.29)   # 报错：2023年不是闰年，2月只有28天
```

**错误提示：**
- `Invalid month: x.` - 无效月份
- `Invalid day: x for month y.` - 无效日期
- `Card for xxxx.xx.xx already exists.` - 卡片已存在

---

### 2. tp - 跳转到指定日期的卡片页面

**格式：**
```
tp(yyyy.m.d)
```

**功能：**
- 跳转到包含指定日期卡片的页面
- 自动滚动视图到该卡片位置

**参数说明：**
- `yyyy`：4位年份
- `m`：月份（1-12）
- `d`：日期（1-31）

**示例：**
```
tp(2024.6.15)   # 跳转到 2024年6月15日 卡片所在页面
```

**错误提示：**
- `Card for xxxx.xx.xx not found.` - 卡片不存在

---

### 3. testcard/tc - 生成测试卡片

**格式：**
```
testcard
tc
```

**功能：**
- 生成日期为 2099.12.31 的测试卡片
- 所有可填写的信息字段都会被适当填充

**填充内容：**
- **天气**：晴，18-26℃，适宜活动
- **睡眠**：23:00 睡着，07:30 自然醒，睡眠质量良好
- **饮食**：早饭：全麦面包、牛奶、鸡蛋；午饭：米饭、清炒西兰花、红烧鱼；晚饭：蔬菜沙拉、糙米、豆腐汤
- **运动**：晨跑30分钟，瑜伽45分钟
- **备注**：测试数据 - BioHacker Editor 功能演示
- **特殊标注**：
  - `[+] 坚持健康生活习惯`
  - `[?] 关注体重变化`
  - `[!] 记得定期体检`

**示例：**
```
testcard   # 创建测试卡片
tc         # 创建测试卡片（缩写形式）
```

**错误提示：**
- `Card for 2099.12.31 already exists.` - 测试卡片已存在

## 通用规则

- 命令不区分大小写，但建议使用小写
- 输入空命令按 Enter 不会执行任何操作
- 未知命令返回：`unknown command.`
- 所有操作结果通过通知系统显示
