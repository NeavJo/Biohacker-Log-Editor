# 控制台命令文档

## 触发方式

在页面中依次按下键盘 `C` → `M` → `D` 键，可打开控制台。

- **打开**：按顺序按下 `C` `M` `D`
- **关闭**：按 `Escape` 键或点击控制台外部
- **发送命令**：输入命令后按 `Enter`

---

## 命令列表

### 1. create - 创建指定日期的卡片

**格式：**
```
create(yyyy.m.d)
```

**参数说明：**
- `yyyy`：4位年份（如 2024）
- `m`：月份（1-12）
- `d`：日期（1-31，需与月份匹配）

**校验规则：**
- 月份必须在 1-12 之间
- 日期必须符合对应月份的天数
  - 1,3,5,7,8,10,12 月：31天
  - 4,6,9,11 月：30天
  - 2 月：平年28天，闰年29天
  - 闰年规则：能被4整除且不能被100整除，或能被400整除

**示例：**
```
create(2024.6.15)   # 创建 2024年6月15日 的卡片
create(2024.2.29)   # 创建 2024年2月29日 的卡片（闰年有效）
create(2023.2.29)   # 报错：2023年不是闰年，2月只有28天
```

**错误提示：**
- `Invalid month: x.` - 无效月份
- `Invalid day: x for month y.` - 无效日期
- `Card for xxxx.xx.xx already exists.` - 卡片已存在

---

### 2. tp - 跳转到指定日期的卡片页面

**格式：**
```
tp(yyyy.m.d)
```

**功能：**
- 跳转到包含指定日期卡片的页面
- 自动滚动视图到该卡片位置

**参数说明：**
- `yyyy`：4位年份
- `m`：月份（1-12）
- `d`：日期（1-31）

**示例：**
```
tp(2024.6.15)   # 跳转到 2024年6月15日 卡片所在页面
```

**错误提示：**
- `Card for xxxx.xx.xx not found.` - 卡片不存在

---

## 通用规则

- 命令不区分大小写，但建议使用小写
- 输入空命令按 Enter 不会执行任何操作
- 未知命令返回：`unknown command.`
- 所有操作结果通过通知系统显示
