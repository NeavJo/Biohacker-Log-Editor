# Biohacker-Log-Editor

这是一个基于 **Vibe Coding**构建的纯前端轻量级内容管理工具（CMS）。

## 🎯 核心核心功能
- **GitHub 数据库化**：免去搭建后端服务器，利用 GitHub REST API 直接读写托管在私有/公开仓库中的 `.txt` 健康日志。
- **智能文本解析 (Parser)**：自动读取规整的纯文本，并利用高容错正则表达式将其转换为现代化、大字号的“可视化日期卡片”。
- **多维度拦截渲染**：针对 `[+]`（有益）、`[?]`（观察）、`[!]`（异常/警告）等生理事件标识进行智能拦截，高亮渲染为彩色标签。
- **一键天气填充**：免 API Key 串联请求 `Open-Meteo` 地理编码与天气预测 API，输入邮编即可一键为今日日志注入气温与天气状态。
- **本地零隐私缓存**：所有敏感配置（GitHub Token、仓库名等）均安全存储于用户本地浏览器（LocalStorage），无任何第三方服务器中转。

## 🛠️ 技术栈
- 前端：HTML5 / Vanilla JavaScript (原生 JS)
- 样式：Tailwind CSS (CDN)
- 数据交互：GitHub REST API (Base64 编解码 + SHA 乐观锁控制)

## 🚀 部署与使用
1. 直接在浏览器中双击打开 `index.html`，或通过 GitHub Pages 在线访问。
2. 点击右上角 **⚙️ 设置**，填写你的 GitHub Token、用户名、仓库名及日志路径。
3. 点击 **加载数据** 开始便捷编辑！
