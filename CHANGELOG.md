# Changelog

## v1.2.1 - 2026-06-08

### Add
- 新增控制台命令 `testcard` 及缩写 `tc`，用于生成日期为 2099.12.31 且各字段均填充的测试卡片

### Changed
- 更新控制台命令文档 [Function/console.md](Function/console.md) 和 README 命令列表

### Fixed
- 修复 Diet 字段午饭冒号后留空时，退出编辑导致午饭字段丢失的 bug；改为午饭为空时显示 `午饭：-`
- 修复 Sleep 编辑模式改为四个独立时间输入框（睡着时/分、醒来时/分）横排排版，搭配自定义切换按钮选择"苏醒"/"闹钟"
- 修复 Diet 编辑模式改为三个独立输入框分别编辑早/午/晚饭，早饭标注（可选），保存后仍保持纯文本格式不变
